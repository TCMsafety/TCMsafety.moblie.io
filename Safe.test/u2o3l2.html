（189）关于出租汽车出车前绕车检查内容，包括 ____。

  1. 车辆外观，包括车灯有无破损，车辆牌号是否清晰  
  2. 轮胎，包括轮胎气压是否正常，有无液体渗漏现象  
  3. 车辆周边环境，包括车辆周围及车底是否有影响车辆正常通行的障碍物  
  4. 车辆卫星定位装置是否完好可用  
     

（190）关于出租汽车安全装置检查，以下说法正确的是 ____。

  1. 安全带伸缩自如，自动锁止功能正常  
  2. 灭火器可忽略有效期，只要外观及压力表完好即可  
  3. 车门处于正常开闭状态，车窗升降功能有效  
  4. 应急报警装置和具有行驶记录功能的车辆卫星定位装置完好有效  
     

（191）依据《出租汽车驾驶员从业资格管理规定》，关于出租汽车驾驶员继续教育内容，包括的是 ____。

  1. 出租汽车安全运营和节能减排知识  
  2. 出租汽车驾驶员的社会责任和职业道德  
  3. 出租汽车驾驶员相关服务规范  
  4. 出租汽车驾驶员从业条件  
     

（192）依据《出租汽车驾驶员从业资格管理规定》，关于出租汽车驾驶员从业资格管理档案，包括的是 ____。

  1. 从业资格考试申请材料  
  2. 继续教育记录和服务质量信誉考核结果  
  3. 驾驶员学历证明  
  4. 违法行为记录、交通责任事故情况  
     

（193）（193） 依据《网络预约出租汽车经营服务管理暂行办法》，关于网约车车辆报废相关规定，下列说法正确的是 ____。

  1. 网约车行驶里程达到60万千米时强制报废  
  2. 网约车车辆使用年限达到8年时强制报废  
  3. 网约车行驶里程未达到60万千米但使用年限达到8年时，退出网约车经营  
  4. 小、微型非营运载客汽车登记为预约出租客运的，按照网约车报废标准报废
