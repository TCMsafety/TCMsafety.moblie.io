（120）依据《巡游出租汽车经营服务管理规定》，在以下选项中，属于巡游出租汽车经营者应当遵守的规定是 ____。

  1. 保证营运车辆性能良好  
  2. 按照国家相关标准运营服务  
  3. 加强从业人员管理和培训教育  
  4. 可将巡游出租汽车交给未经从业资格注册的人员运营  
  5. 在许可的经营区域内从事经营活动，超出许可的经营区域的，起讫点一端应当在许可的经营区域内  
     

（121）依据《网络预约出租汽车经营服务管理暂行办法》，拟从事网约车经营的车辆，应当符合以下条件 ____。

  1. 5座及以下乘用车  
  2. 安装具有行驶记录功能的车辆卫星定位装置  
  3. 车辆技术性能符合运营安全相关标准要求  
  4. 安装应急报警装置  
     

（122）依据《网络预约出租汽车经营服务管理暂行办法》，从事网约车服务的驾驶员，应符合以下哪些条件 ____。

  1. 无交通肇事犯罪，无危险驾驶犯罪记录  
  2. 无暴力犯罪记录  
  3. 无吸毒记录，无饮酒后驾驶记录  
  4. 最近连续3个记分周期内没有记满12分记录  
     

（123）依据《出租汽车驾驶员从业资格管理规定》，申请参加出租汽车驾驶员从业资格考试的，应当符合下列哪些条件？ ____。

  1. 取得相应准驾车型机动车驾驶证并具有2年以上驾驶经历  
  2. 无交通肇事犯罪、危险驾驶犯罪记录，无吸毒记录，无饮酒后驾驶记录，最近连续3个记分周期内没有记满12分记录  
  3. 无暴力犯罪记录  
  4. 城市人民政府规定的其他条件  
     

（124）依据《巡游出租汽车经营服务管理规定》，巡游出租汽车经营者违反本规定，有下列哪些行为 ____，由县级以上地方人民政府出租汽车行政主管部门责令改正，并处以5000元以上1万元以下罚款。构成犯罪的，依法追究刑事责任。

  1. 擅自暂停、终止全部或者部分巡游出租汽车经营的  
  2. 不按照规定保证车辆技术状况良好的  
  3. 不按照规定配置巡游出租汽车相关设备的  
  4. 不按照规定建立并落实投诉举报制度的  
  5. 使用未取得道路运输证的车辆，擅自从事巡游出租汽车经营活动的  
     

（125）根据《出租汽车运营服务规范》中对行车安全的要求，下列说法正确的是 ____。

  1. 遇情况不明、视线不良、起步会车、交叉路口、危险和繁华地段时，应减速慢行  
  2. 遇雨、雾、风沙天气时，应注意路面情况与行人、车辆动态，打开灯光，减速慢行，延长车距，尽量避免超车  
  3. 雪中行车时，宜沿已有车辙低速行驶，避免急加速或急减速  
  4. 通过凹凸不平路段时，应紧握转向盘，低速行驶  
     

（126）根据《网络预约出租汽车运营服务规范》，网络预约出租汽车经营者应建立健全网络预约出租汽车驾驶员岗前培训、继续教育制度，定期组织驾驶员开展有关 ____等方面的教育培训，并建立培训档案。

  1. 法律法规  
  2. 职业道德  
  3. 服务规范  
  4. 安全运营  
     

（127）依据《国务院办公厅关于深化改革推进出租汽车行业健康发展的指导意见》（国办发〔2016〕58号），建立完善以服务质量信誉为导向的经营权配置和管理制度，对经营权期限届满或经营过程中出现 ____等情形的，按有关规定收回经营权。

  1. 重大服务质量问题  
  2. 重大安全生产责任事故  
  3. 严重违法经营行为  
  4. 服务质量信誉考核不合格  
     

（128）依据《网络预约出租汽车经营服务管理暂行办法》规定，拟从事网约车经营的车辆，应当符合以下条件 ____。

  1. 7座及以下乘用车  
  2. 装具有行驶记录功能的车辆卫星定位装置、应急报警装置  
  3. 5座及以下乘用车  
  4. 车辆技术性能符合运营安全相关标准要求  
     

（129）依据《网络预约出租汽车经营服务管理暂行办法》，有下列 ____行为，由县级以上出租汽车行政主管部门和价格主管部门按照职责责令改正，对每次违法行为处以5000元以上10000元以下罚款；情节严重的，处以10000元以上30000元以下罚款。

  1. 未按照规定保证车辆技术状况良好的  
  2. 未按照规定制定服务质量标准、建立并落实投诉举报制度的  
  3. 未履行管理责任，出现甩客、故意绕道、违规收费等严重违反国家相关运营服务标准行为的  
  4. 伪造、变造或者使用伪造、变造、失效的《网络预约出租汽车运输证》《网络预约出租汽车驾驶员证》从事网约车经营活动的  
     

（130）依据《网络预约出租汽车经营服务管理暂行办法》，网约车平台公司应当保证提供服务车辆具备下列哪些条件 ____。

  1. 具备合法营运资质  
  2. 技术状况良好  
  3. 安全性能可靠  
  4. 具有营运车辆相关保险
